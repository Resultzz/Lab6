15 CSS Best Practices

  1. Practice coding everyday.
  2. Use useful websites to help with proper coding.
  3. Use comments to help understand your code.
  4. Use proper indenting while coding to help understand your code.
  5. Ask others for help when doing html/css.
  6. Use books to help with coding.
  7. Practice doing labs and short exercises to remember coding and what they do.
  8. Utilize apps such as Github to help organize html and css documents.
  9. Save your work and refresh your webpage often to understand what your code has accomplished.
  10. You can right-click to inspect your webpage or other webpages to understand how they accomplished their code.
  11. Make use of external style sheets to edit all pages.
  12. Use folders for organization for css, audio, and video files.
  13. Do tests often on your code.
  14. Use shortcuts to make your coding faster.
  15. Use text editors such as Atom or notepad++.
